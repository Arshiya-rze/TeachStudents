در PowerShell این دستورات رو وارد کنید :
1. node -v
2. npm -v
3. ng version
در صورتی که nvm نصب نیست :
4. nvm install 20
5. nvm use 20
6. node -v
حذف CLI جهانیِ Angular 20 و نصب Angular 19
7. npm uninstall -g @angular/cli
پاک سازی کش سیستم
8. npm cache verify
9. npm i -g @angular/cli@19
10. ng version
