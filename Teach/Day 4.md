کاربردهای HTML  

کاربرد اصلی HTML این است، که با استفاده از تگ‌ها(Tag) قالب اصلی صفحات وب را پیاده سازی میکند. در اصل اسکلت اولیه صفحات وب با استفاده از تگ‌های HTML صورت میگیرد. هر کدام از این تگ‌ها کاربردهای خاص خود را دارند و باعث نمایش دقیق عناصر در صفحات وب می شوند.
درسته که با زبان های مختلف ما کد میزنیم ممکنه مه افرادی React ویا Angular کد بزن خود ما  از انگولار استفاده میکنیم ولی در آخر همه ی این ها تبدیل میشن به HTML چون مرورگر فقط زبان Html, Css میشناسه نه چیز دیگه ای.

### HTML Tutorial
مثال های پایه HTML   
در طول آموزش اگر از برچسب هایی استفاده  میکنیم که هنوز درباره آنها یاد نگرفته اید نگران نباشید. 
اسناد HTML  
همه اسناد HTML باید با یک اعلان نوع سند شروع شوند که آن را با تگ <!DOCTYPE> ایجاد میکنیم. 
خود سند HTML با تگ <html> شروع و با تگ </html> به پایان میرسد. 

قسمت قابل مشاهده یک سند HTML بین تگ <body> و </body> قرار میگیرد. 

همانطور که در این دو مورد مشاهده کردید تمامی تگ های HTML (به جز موارد استثنا) یک شروع و پایانی دارند به طور مثال: 

<a> 

. 

. 

</a>  

این قطعه کد مثالی برای ایجاد یک سند HTML است: 

<!DOCTYPE html> 

<html> 

<body> 

<h1>My First Heading</h1> 

<p>My first paragraph.</p> 

</body> 

</html> 

اعلان <!DOCTYPE> نوع سند را نشان می دهد و به مرورگرها کمک می کند تا صفحات وب را به درستی نمایش دهند. 

فقط باید یک بار، در بالای صفحه (قبل از هر تگ HTML) ظاهر شود. 

این اعلان به کوچک یا بزرگ بودن حروف حساس نیست. 

اعلان <!DOCTYPE> برای HTML5 : <!DOCTYPE html>  

در بین تگ <body></body> سه تگ زیر قرار دارند: 

تگ <header></header> برای ایجاد هدر (بالای صفحه)، 

تگ <main></main> برای ایجاد قسمت اصلی (وسط صفحه)، 

تگ <footer></footer> برای ایجاد فوتر (پایین صفحه). 

سرفصل های HTML 

سرفصل های HTML با تگ های <h1> تا <h6> تعریف می شوند. 

<h1> مهمترین عنوان را تعریف می کند. <h6> کم اهمیت ترین عنوان را تعریف می کند 

مثال: 

<h1>This is heading 1</h1> 

<h2>This is heading 2</h2> 

<h3>This is heading 3</h3> 

پاراگراف های HTML 

پاراگراف های HTML با تگ <p> تعریف می شوند 

مثال: 

<p>This is a paragraph.</p> 

<p>This is another paragraph.</p> 

لینک های HTML 

لینک ها در HTML با تگ <a> تعریف می شوند 

مثال: 

<a href="https://www.w3schools.com">This is a link</a> 

لینک را در ویژگی (href) مشخص میکنیم، 

برای ارائه اطلاعات اضافی در مورد عناصر HTML یک سری ویژگی وجود دارد که در آینده تعداد بیشتری از آن ها خواهید آموخت. 

تصاویر HTML 

تصاویر در HTML با تگ <img> تعریف می شوند. 

فایل منبع (src)، متن جایگزین (alt) و همچنین (width و height) به عنوان ویژگی ارائه شده است 

مثال:  

<img  src="w3schools.jpg" alt="W3Schools.com" width="104" height="142"> 

نکته: 

"تگ <img> از جمله تگ هایی هست که به صورت تکی نوشته میشود" 

نحوه مشاهده منبع HTML 

آیا تا به حال یک صفحه وب رو دیدید یا براتون سوال پیش اومده که چطور این کار رو انجام دادن! 

مشاهده کد منبع HTML 

کلیدهای ترکیبی CTRL + U را در یک صفحه HTML بزنید، یا روی صفحه کلیک راست کرده و "View Page Source" را انتخاب کنید. با این کار یک تب جدید حاوی کد منبع HTML صفحه باز می شود. 

یک عنصر HTML را بررسی کنید 

برای این کار روی یک عنصر (یا یک منطقه خالی از صفحه) کلیک راست کرده و "Inspect" را انتخاب کنید تا ببینید از چه عناصری تشکیل شده است (هم HTML و هم CSS را خواهید دید). 

همچنین می‌توانید HTML یا CSS را از این قسمت در پانل عناصر یا سبک ‌هایی که باز می‌شود ویرایش کنید.**

بازارکار و درآمد HTML و CSS 

آینده‌ای روشن برای طراحی صفحات با HTML و CSS وجود دارد، به این دلیل که صفحات وب رشد و پیشرفت‌های چشمگیری در سال‌های اخیر داشته است. اینترنتی شدن بیشتر کسب و کارها، باعث رقابت شدید بین توسعه دهندگان، طراحان و برنامه نویسان شده است. بستر‌کاری زیادی برای برنامه نویسان و طراحان وب وجود دارد و در آینده این روند پیشرفت خواهد کرد.

میزان درآمد یک برنامه نویس وب در پروژهای ایرانی، بین 6 الی 40 میلیون بوده و درآمد پروژه‌های خارجی حدود 60 هزار دلار الی 75 هزار دلار می باشد. این مبالغ کاملا رقابتی هستند و ممکن است نسبت به پروژه و تخصص شما تغییر کنند. نکته مهم این است که حتما باید در کنار HTML و CSS به یک زبان برنامه نویسی، فریم ورک یا کتابخانه تسلط داشته باشد. بازار کار HTML و CSS بیشتر نیاز به نیروی متخصص و کارآمد دارد

**HTML Styles - CSS**
**CSS مخفف عبارت Cascading Style Sheets است. 

CSS مقدار زیادی از کار را ذخیره می کند. این می تواند طرح بندی چندین صفحه وب را به طور همزمان کنترل کند. 

CSS = Styles and Colors 

Manipulate Text 

Colors,  Boxes 

CSS 

 (CSS) برای قالب بندی طرح بندی یک صفحه وب استفاده می شود. 

  

با CSS می ‌توانید: 

رنگ 

نوع نگارش متن (فونت) 

اندازه متن 

فاصله بین عناصر 

نحوه قرارگیری و چیدمان عناصر 

تصاویر پس ‌زمینه یا رنگ ‌های پس ‌زمینه مورد استفاده نمایشگر های مختلف 

برای دستگاه‌ ها و اندازه ‌های مختلف صفحه و خیلی موارد بیشتری را کنترل کنید. 

نکته: 

 کلمه Cascading به این معنی است که سبکی که برای یک عنصر parents اعمال می شود برای همه عناصر children در آن نیز اعمال می شود. 

بنابراین، اگر رنگ متن در تگ <body> را روی "آبی" تنظیم کنید، همه سر فصل ها، پاراگراف ها و سایر عناصر متن داخل بدنه نیز یک رنگ خواهند داشت (مگر اینکه چیز دیگری را مشخص کنید)! 

(البته این امکان آموزش خواهیم داد که هر متن child را به رنگ مخالف parent تغییر دهیم) 

اضافه کردن CSS به فایل های HTML  

CSS را می توان به سه روش به فایل های HTML اضافه کرد: 

Inline (در خط): با استفاده از ویژگی style در عناصر HTML. 

Internal  (داخلی): با استفاده از تگ <style> در بخش <head>. 

External (خارجی): با استفاده از تگ <link> برای لینک دادن به یک فایل CSS خارجی. 

رایج ترین راه برای افزودن CSS، نگه داشتن styles در فایل های CSS خارجی است. (هر سه روش را آموزش خواهید دید).

Inline CSS (درون خطی) 

یک CSS درون خطی برای اعمال یک سبک منحصر به فرد به یک عنصر HTML استفاده می شود. 

یک CSS درون خطی از ویژگی style یک عنصر HTML استفاده می کند. 

مثال زیر رنگ متن عنصر <h1> را آبی و رنگ متن عنصر <p> را قرمز تنظیم می کند: 

<h1 style="color: blue;">A Blue Heading</h1> 

<p style="color: red;">A red paragraph.</p> 

Internal CSS (داخلی) 

یک CSS داخلی برای تعریف یک سبک برای کل یک صفحه HTML استفاده می شود. 

CSS داخلی در بخش <head> یک صفحه HTML در یک تگ <style> تعریف شده است. 

در مثال زیر رنگ متن همه عناصر <h1> (در آن صفحه) را آبی و رنگ متن همه عناصر <p> را قرمز تنظیم می کند. 

علاوه بر این، صفحه با رنگ پس زمینه "آبی پودری" نمایش داده می شود: 

<!DOCTYPE html> 

<html> 

<head> 

<style> 

body {background-color: powderblue;} 

h1   {color: blue;} 

p    {color: red;} 

</style> 

</head> 

<body> 

<h1>This is a heading</h1> 

<p>This is a paragraph.</p> 

</body> 

</html> 

External CSS (خارجی) 

یک فایل خارجی به منظور تعریف سبک برای بسیاری از صفحات HTML استفاده می شود. 

برای استفاده از یک فایل خارجی، باید یک لینک به آن در بخش <head> هر صفحه HTML اضافه کنیم، به طور مثال: 

<!DOCTYPE html> 

<html> 

<head> 

  <link rel="stylesheet" href="styles.css"> 

</head> 

<body> 

<h1>This is a heading</h1> 

<p>This is a paragraph.</p> 

</body> 

</html> 

  

فایل خارجی را می توان در هر ویرایشگر متنی نوشت. فایل نباید حاوی هیچ کد HTML باشد و باید با پسوند css. ذخیره شود. 

در این مثال کد زیر یک "styles.css" است که کل صفحه را به رنگ پس زمینه "آبی پودری" ، رنگ متن همه عناصر <h1> را آبی و رنگ متن همه عناصر <p> را قرمز تنظیم می کند: 

body { 

  background-color: powderblue; 

} 

h1 { 

  color: blue; 

} 

p { 

  color: red; 

}

CSS Border  

ویژگی Border   حاشیه ای را در اطراف یک عنصر HTML تعریف می کند. 

نکته:  

("می توانیم برای تقریباً تمام عناصر HTML یک حاشیه تعریف کنیم.") 

به عنوان مثال، کد زیر برای تمامی عناصر تگ <p> یک حاشیه ایجاد میکند: 

p { 

  border: 2px solid powderblue; 

} 

CSS Padding 

ویژگی  padding یک (فضا) بین متن و حاشیه را ایجاد می کند. 

مثال: برای ایجاد حاشیه و فضا: 

p { 

  border: 2px solid powderblue; 

  padding: 30px; 

} 

CSS Margin 

ویژگی margin یک (فضا) را در خارج از حاشیه ایجاد می کند. 

مثال: برای ایجاد حاشیه خارج از فضا:  (بین دو border) 

p { 

  border: 2px solid powderblue; 

  margin: 50px; 

} 

Link to External CSS  

فایل های CSS خارجی را می توان با یک URL کامل یا با یک مسیر، نسبت به صفحه وب فعلی ارجاع داد. 

این مثال از یک URL کامل برای پیوند دادن به یک فایل CSS خارجی استفاده می کند: 

<head> 

   <link rel="stylesheet" href="https://www.w3schools.com/html/styles.css"> 

</head> 

این مثال به یک فایل CSS واقع در پوشه HTML در وب سایت فعلی لینک ایجاد میکند: 

<head> 

   <link rel="stylesheet" href="/html/styles.css"> 

</head> 

این مثال به یک فایل CSS که در همان پوشه صفحه فعلی قرار دارد لینک ایجاد میکند: 

<head> 

   <link rel="stylesheet" href="styles.css"> 

</head>**
